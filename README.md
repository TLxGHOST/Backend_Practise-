# Backend Learning Modules

##  Introduction
These modules helped me build a **strong backend foundation** and understand how data flows between the **client and server**.

They are **not part of a single project**. Instead, they are small learning modules created to help beginners understand different backend concepts.

Due to a tight schedule, I might not be able to provide very detailed explanations for every module, but the code should still give you a good idea of how things work.

---

##  Running the Modules

Install the required dependencies:

```bash
npm install
```

Some modules may also require **EJS**, so install it if needed:

```bash
npm install ejs
```
And finally use :

```bash
npx nodemon filename.js
```
***filename*** is in the server side and just open the port mentioned in the file on local machine (in general PORT 3000 has been used😅😅)

---

##  Database

The database used in some modules is:

**PostgreSQL**

 Important:
- These modules use a **local database setup**
- They are mainly provided as **reference templates**
- Since the database runs locally on my system, it will **not connect directly for you**

You will need to **configure your own PostgreSQL instance** to run them.

## ⚠️ Important Note

In some database and configuration files, **passwords and certain details are left exposed**.

This is **not good security practice**, but since these modules are meant only for **learning and experimentation**, please treat them as examples rather than production-ready code.
